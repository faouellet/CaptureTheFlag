Installation:

1- Rendez vous à http://aisandbox.com/download/ et choissisez votre plateforme

2- Télécharger et installer AI Sandbox 

Exécution:

1- Ouvrir CaptureTheFlag.sln

2- Compiler

3- Lancer le projet de tests

À noter que certains tests ne passent pas, car le code qu'ils vont tester n'a pas encore été écrit.
L'exécution du projet va donc s'arrêter en mi-parcours. Le but est de présenter le "framework" de
tests conçus pour la suite du projet.
